# 📝 Express Blog App

A simple blog application built using **Node.js**, **Express.js**, **EJS**, and **CSS**. This project was created to understand how CRUD operations work in a web application and how the frontend and backend communicate with each other.

Users can create, view, edit, and delete blog posts through a clean and easy-to-use interface.

## ✨ Features

- Create a new blog post
- View all blog posts
- Edit existing posts
- Delete posts
- Responsive and clean user interface
- Server-side rendering using EJS

## 🛠️ Built With

- Node.js
- Express.js
- EJS
- HTML5
- CSS3
- JavaScript

## 🚀 Getting Started

1. Clone this repository

```bash
git clone https://github.com/your-username/Express-Blog-App.git
```

2. Open the project folder

```bash
cd Express-Blog-App
```

3. Install dependencies

```bash
npm install
```

4. Start the server

```bash
nodemon index.js
```

or

```bash
node index.js
```

5. Open your browser and visit

```
http://localhost:3000
```
